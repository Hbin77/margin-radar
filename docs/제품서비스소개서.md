## 1. 제안 배경 및 출품작 소개

### 1-1. 문제: 공공조달 시장은 셀러에게 '깜깜이'다
- 공공조달 시장 규모는 연 200조 원 이상. 그러나 중소 제조사·수입사·온라인 셀러에게 이 시장은 **정보 비대칭의 벽**에 막혀 있다.
- 셀러는 세 가지를 모른다: ① 정부가 **무엇을 얼마에** 사는지, ② 그 품목의 **경쟁 강도**, ③ **어떤 가격·경로**로 진입해야 하는지.
- 결과: "공공조달은 어렵다"는 인식만으로 다수 셀러가 **시도조차 하지 않는다.**

### 1-2. 출품작: MR(Margin Radar)
조달 데이터로 **"정부 구매가 활발하고 · 경쟁이 적으며 · 시중가 대비 마진이 있는" 품목**을 찾아, 셀러·수입사에게 진입 우선순위를 제시하는 **공공조달 소싱 인텔리전스 엔진.**

- **두 얼굴 한 엔진**: 같은 분석이 ① 셀러에겐 *고마진 판로 발굴*(비즈니스), ② 사회엔 *조달 가격·정보 투명성*(공공가치)을 동시에 제공.

## 2. 핵심 내용 (요약)

- **블루오션 점수**: 마진배수(30%) + **진입 가능성(30%)** + **수요(25%)** + 저경쟁도(15%)를 결합한 0~100 점수. (수요 = 종합쇼핑몰 납품요구 실거래 집계)
- **진입 가능성(핵심 차별점)**: 품목이 「중소기업자간 경쟁제품」으로 지정됐는지 판별 → *고마진이지만 중소기업 직접생산자만 납품 가능한 '막힌 시장'* 과 *일반 제조·수입사도 진입 가능한 '개방 시장'* 을 구분. "마진 = 기회"의 함정을 데이터로 거른다.
- **3개 화면**: ① 블루오션 랭킹 ② 품목 상세(경쟁업체·단가분포·규격·실물이미지·규격서 링크) ③ 내 제품 역조회.
- **데이터**: 조달청 종합쇼핑몰 품목정보 API(단가·업체·규격·이미지·중기경쟁여부) + 네이버 쇼핑(시중가) 결합.
- **AI 모델**: 조달 45만 건 기반 **가격예측 앙상블(HistGBM+RandomForest→Ridge 스태킹, R²=0.948)** 로 조달 적정가 예측 → **out-of-fold 잔차** 기반 **이상(과지출) 탐지**, **KMeans 시장 세분화**, **IsolationForest 이상거래 탐지**.

### 3-2. 데이터 활용 방법 / 분석 기법
1. **수집**: 조달청 API를 품명(prdctClsfcNoNm) × 등록일 12개월 윈도로 스윕, 페이지네이션·중복제거.
2. **정제**(검증 반영): 단위 오염 제거(개/대만), 단가 p5~p95 트림(이상치 제거), 비제품(서비스·임대·부품) 품명 제외.
3. **매칭**: 품명 기반 네이버 검색 후 **'정부단가 40% 이상' 동급 필터**로 가정용 저가품을 배제(등급 보정).
4. **진입 가능성 판별**: 품목별 「중소기업자간 경쟁제품」 비율 산출 → 개방형(<30%)·혼합(30~70%)·중기경쟁 제한(>70%) 분류.
5. **수요 집계**: 종합쇼핑몰 납품요구 실거래(1개월 21.6만 건)를 품명별로 집계 → 정부 구매액·거래건수·수요기관수.
6. **스코어링**: 블루오션점수 = 0.30×마진 + 0.30×진입가능성 + 0.25×수요 + 0.15×저경쟁. (진입 막힌 시장은 마진이 높아도 강등, 수요 큰 개방시장이 상위)

### 3-3. 데이터의 지속적 확보 가능성
- 조달청 API는 전일(D-1) 자동 갱신·무료(개발계정 발급 완료), 네이버 검색 API 일 25,000건. **상시 자동 수집 파이프라인 구축 가능.**

### 3-3-A. AI 모델 (핵심 기술)
- **목표/레이블**: 관측된 계약단가(log) — 45.5만 행 수집(속도 위해 학습 표본 15만 상한)(블루오션 점수를 직접 학습하는 순환을 피하고, 실측 가격이라는 진짜 레이블 사용).
- **피처**: 세부품명·제조사·단위·기업구분·인증플래그 + 규격 텍스트에서 파싱한 치수/부피/수치. 고차원 범주는 TargetEncoder(CV, 누수 방지). 타깃 누수 점검 완료(가격 파생값 미사용).
- **모델**: HistGradientBoosting + RandomForest를 Ridge로 스태킹한 앙상블.
- **성능(공정 비교)**: 홀드아웃 **앙상블 R²=0.948**(HGB 0.941), MAPE 21.0%. 단순 비교가 아니라 **'세부품명 룩업' 베이스라인(R²=0.840)을 MAE 33% 추가 개선** → 규격·제조사 등 카테고리 내 변동까지 학습(룩업 이상의 가치 입증). 일반화 검증: 미지 카테고리 GroupKFold R²=0.41(원리적 한계 고지).
- **해석**: 피처 중요도 — 세부품명(0.82) > 제조사 > 부피·규격 (조달 단가 결정요인 규명).
- **활용**: ① 예측 적정가 vs 실제가 = **과지출 이상탐지**(out-of-fold 잔차, 감사·공공가치) ② **KMeans 시장 세분화**(블루오션/관망/진입제한) ③ **IsolationForest 이상거래 탐지**.
- **정직성**: 랭킹 공식은 투명 유지(마진·진입·수요·경쟁), ML은 가격·이상·세분화의 **독립 신호 레이어**로 결합(블랙박스화 회피).

### 3-4. 시장성 및 창업 계획
- 1차 고객: 공공조달 진입을 노리는 중소 제조사·수입사·온라인 셀러(지너스인터내셔널 본업과 직결, 내부 소싱 도구로 즉시 활용).
- 수익모델: 구독(품목 모니터링·알림), 심화 리포트, B2B API.
- 로드맵: 카테고리 확장 → 수요(거래량) 축 추가 → 입찰·납품요구 연계 → 진입 컨설팅.

### 3-5. 개발 과정 및 방법 / 개발 툴
- 데이터 파이프라인: Python(표준 라이브러리) — 수집·정제·스코어링.
- 웹: 정적 SPA(HTML/CSS/JS) + 디자인 토큰 시스템(WCAG AA 검증), 정적 호스팅 배포.

## 4. 결과 및 기대효과

### 4-1. 기존 서비스와의 차별성
- 기존 입찰분석 서비스(비드 등)는 *입찰을 돕는다*. MR은 **"애초에 무엇을 팔지"를 역으로 발굴**한다 — 조달 데이터를 *감시/낙찰*이 아닌 **소싱 기회**로 재정의한 최초 시각.
- 조달청 자체 수요예측과 달리, **시장(시중가) 결합 + 셀러 관점**.
- **"고마진 ≠ 기회"를 거르는 유일한 도구**: 마진만 보는 분석은 컴퓨터책상(2.27배)을 추천하지만, MR은 그것이 중소기업자간 경쟁제품(97%)으로 일반 셀러가 진입 불가임을 판별해 강등하고, **진입 가능한 개방 시장(모니터·복사기·진공청소기 등)을 우선 추천**한다. 이 진입 가능성 판별이 실사용 가치의 핵심.

### 4-2. 정직한 검증 (방어 가능한 수치)
- 초기 단순 비교는 "정부가 시중가 5배에 산다"는 과장을 낳았다 → **적대적 재검증**으로 그 격차의 상당부분이 *기관용 vs 가정용 등급 차이*임을 규명.
- **진입 가능성으로 재정렬한 결과(핵심)**: 마진만 보면 컴퓨터책상(2.26배)이 1위지만 중기경쟁 97%로 막힌 시장. **진입 가능한 개방 시장**을 우선하면 — 레이저프린터 2.09배(경쟁 7곳)·오디오모니터 1.82배·진공청소기 1.73배·공기청정기 1.65배 등이 상위. 이게 지너스가 실제로 들어갈 수 있는 기회.
- 데이터 규모: 조달청 종합쇼핑몰 **45.2만 건**(3개 계약유형, 829 품명) 수집·정제 → **256개 품목** 랭킹(개방형 151). ML 가격예측 R²=0.948. **매일 자동 갱신**(cron).
- 메시지: 자극적 과장이 아니라 **"동급 기준으로도 존재하는 가격·정보 격차 + 셀러의 미인지"** = 진짜 기회.

### 4-3. 사회적 가치 (효과성)
- 중소·창업기업의 공공조달 **진입장벽 완화**(정책 부합), 조달 가격·정보 **투명성 제고**.

2. 매칭은 키워드+가격필터+제목관련성(하이브리드) 수준 → 제조사·모델·규격 단위 LLM 매칭으로 고도화 여지.
3. 수요는 1개월(2026-05-15~06-14) 실거래 스냅샷 — 계절성 보정 위해 6~12개월 누적이 다음 단계.
